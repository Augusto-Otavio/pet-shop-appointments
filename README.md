<h1 align="center">PetShop Scheduler</h1>

<p align="center">
  <strong>Uma aplicação de agendamento de serviços para petshop, com interface moderna e integração com API.</strong>
</p>

<p align="center">
  Este projeto foi desenvolvido como parte do curso de Desenvolvedor Full Stack da
  <a href="https://www.rocketseat.com.br/" target="_blank">Rocketseat</a>,
  com o objetivo de aplicar conceitos importantes de JavaScript moderno,
  consumo de API e organização de código frontend.
</p>

<h2>Funcionalidades</h2>

<h3>Agenda de Atendimentos</h3>
<ul>
  <li>Visualização da agenda diária</li>
  <li>Organização por períodos: manhã, tarde e noite</li>
  <li>Horários bem definidos para cada atendimento</li>
  <li>Identificação do pet e do serviço agendado</li>
</ul>

<h3>Novo Agendamento</h3>
<ul>
  <li>Nome do tutor</li>
  <li>Nome do pet</li>
  <li>Telefone para contato</li>
  <li>Descrição do serviço (banho, tosa, consulta, etc.)</li>
  <li>Seleção da data do atendimento</li>
  <li>Escolha do horário disponível</li>
</ul>

<h3>Modal de Agendamento</h3>
<ul>
  <li>Formulário exibido em modal</li>
  <li>Bloqueio da tela de fundo durante o agendamento</li>
  <li>Criação de novos atendimentos via API</li>
</ul>

<hr>

<h2>Interface e Experiência do Usuário</h2>
<ul>
  <li>Design em modo escuro</li>
  <li>Destaque visual para ações principais</li>
  <li>Layout focado na leitura clara dos horários</li>
  <li>Navegação simples e intuitiva</li>
</ul>

<hr>

<h2>Tecnologias Utilizadas</h2>

<h3>HTML5</h3>
<ul>
  <li>Estrutura semântica da página</li>
  <li>Criação de formulários acessíveis</li>
  <li>Uso correto de <code>label</code>, <code>input</code> e <code>button</code></li>
</ul>

<h3>CSS3</h3>
<ul>
  <li>Estilização moderna com foco em dark theme</li>
  <li>Layouts utilizando <code>flexbox</code></li>
  <li>Componentização visual da agenda</li>
</ul>

<h3>JavaScript</h3>
<ul>
  <li>Organização do código utilizando ES Modules</li>
  <li>Consumo de API simulada com JSON Server</li>
  <li>Requisições HTTP para listar, criar e consultar agendamentos</li>
  <li>Leitura e manipulação de dados vindos da API</li>
  <li>Uso da biblioteca Day.js para manipulação e formatação de datas e horários</li>
  <li>Manipulação do DOM e controle do modal</li>
</ul>

<p><strong>Projeto com API simulada, sem persistência em produção.</strong></p>
